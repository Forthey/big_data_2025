1. **Чем отличаются друг от друга постановки следующих задач машинного обучения: обучение по прецедентам, обучение с учителем, обучение без учителя?**

   - **Обучение по прецедентам** - это наиболее общая парадигма машинного обучения, в рамках которой на основе имеющихся примеров требуется выявить закономерности.  
   - **Обучение с учителем** - подход, при котором используются размеченные данные вида «вход - правильный выход», и модель обучается находить соответствие между ними.  
   - **Обучение без учителя** - подход, в котором имеются только входные данные без готовых ответов, и задача состоит в выявлении скрытых закономерностей или структуры.

2. **Приведите примеры ответов (откликов), которые можно отнести к булевым, номинальным, порядковым и количественным. Свяжите тип отклика с видом функции потерь.**

   - **Булевый отклик** - двоичное значение (0 или 1), например, «спам / не спам». Связан с функцией потерь: бинарная кросс-энтропия, которая штрафует не только за ошибку, но и за низкую уверенность модели.  
   - **Номинальный (категориальный) отклик** - принадлежность к одному из нескольких несвязанных классов (например, «кошка», «собака», «птица»). Используется категориальная кросс-энтропия.  
   - **Порядковый отклик** - значение из ранжированных категорий (например, оценка от 1 до 5). Может применяться взвешенная версия кросс-энтропии с усилением штрафа при большем отклонении от истинного класса.  
   - **Количественный отклик** - непрерывное числовое значение (например, цена товара). Используются функции потерь: среднеквадратическая ошибка (MSE) или средняя абсолютная ошибка (MAE).

3. **Другие примеры функций потерь**

   - Один из примеров - **Adversarial Loss**, применяемая при обучении генеративно-состязательных сетей (GAN). Её суть в использовании дополнительной модели (дискриминатора), которая оценивает реалистичность сгенерированных данных, и на основе этой оценки формируется функция потерь для генератора.
